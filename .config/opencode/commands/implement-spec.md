# Implement specification

If no specification was provided, abort and ask the user to create a specification first.

Copy this checklist and track progress:

- [ ] Phase 1: Divide specification into independent parts
- [ ] Phase 2: For each part:
  - [ ] Code: Implement the code
  - [ ] Tests: Write tests
  - [ ] Validate: Verify against spec
  - [ ] Verify: Run tests, lint, format (max 3 attempts per part)
- [ ] Phase 3: Verify complete implementation against full specification
